# Flower Lab

Flower Lab — каталог букетів. Сучасний вебзастосунок для перегляду та замовлення квіткових композицій.

## Структура проекту

```
flowerlab/
├── apps/
│   ├── frontend/         ← Next.js (React)
│   └── backend/          ← Node.js + Express
├── .env.example
├── docker-compose.yml
└── README.md
```

## Стек

- **Frontend:** Next.js, React
- **Backend:** Node.js, Express
- **БД:** PostgreSQL
- **Локальна розробка:** Docker Compose
